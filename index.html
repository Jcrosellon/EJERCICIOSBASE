Ejercicio #2

Listado de departamentos registrados en la base de datos

  • Nombre del departamento
  • Nombre del pais
SELECT d.name AS department_name, c.name AS country_name FROM departments d JOIN countries c ON d.countries_id = c.id;

Ejercicio #3

Listado de escuelas por un pais predeterminado

  • Nombre de la escuela
  • Nombre del pais
SELECT s.name AS school_name, c.name AS country_name FROM schools s JOIN countries c ON s.countries_id = c.id WHERE c.id = (SELECT id FROM countries WHERE name = 'colombia');

Ejercicio #4

Listado de usuarios ordenados de mayor edad a menor edad

  • Nombres
  • Apellidos
  • Fecha de nacimiento
SELECT names, lastname, date_birth FROM users ORDER BY date_birth DESC;

Ejercicio #5

En que departamento hay mas usuarios registrados

  • Departamento
  • ID
SELECT d.name AS department_name, COUNT(u.id) AS total_users FROM departments d JOIN users u ON d.id = u.departments_id GROUP BY d.name ORDER BY total_users DESC LIMIT 1;

Ejercicio #6

Cual es el pais con mas usuarios registrados

  • Pais
  • ID
SELECT c.name AS country_name, COUNT(u.id) AS total_users FROM countries c JOIN departments d ON c.id = d.countries_id JOIN users u ON d.id = u.departments_id GROUP BY c.name ORDER BY total_users DESC LIMIT 1;

Ejercicio #7

Listar solo 3 departamentos

  • Nombres
  • Departamento
  • ID
SELECT name AS department_name, id FROM departments WHERE id BETWEEN 1 AND 3;

Ejercicio #8

Listar la informacion de un usuario determinado

  • Nombres
  • Apellidos
  • Correo
  • Nombre del colegio
  • nombre del departamento
  • nombre del Pais
SELECT u.names, u.lastname, u.email, s.name AS school_name, d.name AS department_name, c.name AS country_name FROM users u JOIN schools s ON u.schools_id = s.id JOIN departments d ON u.departments_id = d.id JOIN countries c ON d.countries_id = c.id WHERE u.id = '10';

Ejercicio #9

Ejercicio #10

Cuantos usuarios viven en un departamento determinado

  • Cantidad de usuarios

Ejercicio #11

Listado de usuarios que tengan mas de 20 años

  • Nombres
  • Apellidos
  • Fecha de nacimiento
SELECT names, lastname, date_birth FROM users WHERE date_birth <= DATE_SUB(CURDATE(), INTERVAL 20 YEAR);

Ejercicio #12

Listado de usuarios que tengan de 20 a 60 años

  • Nombres
  • Apellidos
  • nombre capital
SELECT users.names, users.lastname, countries.capital FROM users JOIN departments ON users.departments_id = departments.id JOIN countries ON departments.countries_id = countries.id WHERE TIMESTAMPDIFF(YEAR, users.date_birth, CURDATE()) BETWEEN 20 AND 60;

Ejercicio #13

Listado de colegios con el numero 7 en su telefono

  • Nombre del colegio
  • telefono
  • ID
SELECT name, address, phone FROM schools WHERE phone LIKE '%7%';
